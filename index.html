How the Pipeline Works

1. Raw Ingest (GCP)

Cloud Storage & Eventarc

High-res mezzanine files and sidecars (audio, subtitles) are uploaded to a Google Cloud bucket. Eventarc instantly detects the upload and triggers a serverless Cloud Run container to validate the bundle.

2. Transcode (AWS)

MediaConvert & Boto3

The Python orchestrator generates secure, short-lived IAM Signed URLs. AWS MediaConvert pulls the GCP assets across the cloud boundary and crunches them into a multi-bitrate HLS ladder (1080p, 720p, 480p).

3. Delivery & Auth

S3 & Secure Shaka Player

Packaged `.m3u8` and `.ts` files land in an S3 bucket. To play them, the client player must inject cryptographic auth tokens into the network header of every chunk request, stopping direct URL piracy.
