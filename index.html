How The Pipeline Works

1

Raw Ingest

GCP Cloud Storage

High-res mezzanine files and sidecars (audio, subtitles) are uploaded. This triggers a serverless Cloud Run container to validate the assets.

2

Transcoding

AWS MediaConvert

The GCP orchestrator generates signed URLs for the raw assets and submits a job to AWS to generate an adaptive bitrate (ABR) ladder.

3

Delivery

AWS S3 & Client Auth

AWS outputs HLS segments. Shaka Player intercepts network requests in the browser to inject custom tokens for secure playback.
